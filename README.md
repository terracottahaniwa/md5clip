# md5clip
<p>
put md5 digest to clipboard.<br>
clipboard will cleared when after 10sec.<br>
</p>

### install and usage:
<p>
clone or download zip from this repo. (and decompress zip)<br>
execute ./dist/md5clip/md5clip.exe<br>
</p>
